## 做了什么

### 1. Critical Bug Fix: DEFAULT_CONSTRAINTS_DIR

Discovered and fixed a path traversal bug in `constraint_parser.py`:
- **Bug**: `Path(__file__).parent.parent.parent.parent` (4 levels) resolved to `/home/yan/clawd/` instead of `/home/yan/clawd/nous/`
- **Impact**: Any code using the default path loaded **0 constraints** — gate ran as pure allow-all
- **Affected**: Live `gateway_hook.py` (uses default), `run_challenge_benchmark.py` (new)
- **Not affected**: `run_split_benchmark.py` (passes explicit path via `CONSTRAINTS_DIR`)
- **Fix**: Changed to `.parent.parent.parent` (3 levels), verified resolves to correct path
- **Safety improvement**: Added loud warnings when 0 constraints load (Gemini critique)

This bug was latent since the constraint system was introduced. All existing benchmarks masked it by passing explicit paths.
