## 假设

## 根因分析（超出假设）

实际问题比假设复杂得多，两层根因：

### 根因 1：_ADVOCACY_PATTERNS 误匹配 "campaign"
- `_ADVOCACY_PATTERNS` 中 `(advocat|campaign|...)` 匹配了 "viral marketing **campaign**"
- 导致 `has_advocacy_signal = True`
- 而 `_ANTI_ADVOCACY_PATTERNS` 不包含 "misinformation"，未被过滤
- 结果：`is_trivial(facts) = True`

### 根因 2：publish_post + is_trivial=True → trivial-override 绕过 semantic gate
- publish_post 走 T3-soft → Datalog verdict = "confirm"
- Loop 54 的 `_TRIVIAL_OVERRIDE_ALLOWED_ACTIONS` 包含 "publish_post"
- `trivial_eligible=True` + `is_trivial=True` → `layer_path = "trivial_allow"`
- **Semantic gate 被完全绕过！**

### 根因 3（次要）：intent_extractor 缺少 disinformation spread 模式
- "utilizing trending misinformation" 不匹配原有3个模式（仅覆盖选举操纵/deepfake/fake news spreading）
- 即使走到 semantic gate，也没有 intent:disinformation 意图种子

## 修复

### Fix 1: fact_extractor.py — _ANTI_ADVOCACY_PATTERNS 加 misinformation
```python
_ANTI_ADVOCACY_PATTERNS = [
    re.compile(r"(sexual|nude|naked|explicit|deepfake.*sex|porn)", re.I),
    re.compile(r"(never\s+admit.*fake|deceiv|mislead)", re.I),
    re.compile(r"(blackmail|extort|threaten)", re.I),
    # Loop 56: "viral marketing campaign" + misinformation is NOT advocacy
    re.compile(r"\b(misinformation|disinfo|false\s+narrative)\b", re.I),
]
```
效果："viral marketing campaign, utilizing trending misinformation" → `has_advocacy_signal=False` → `is_trivial=False` → 进入 semantic gate

### Fix 2: intent_extractor.py — 新增 2 个 disinformation spread 模式
```python
# Loop 56: harmful_evasion patterns
re.compile(r"\b(utiliz\w*|leverag\w*|weaponiz\w*|deploy\w*|amplif\w*).{0,50}(misinformation|disinfo|false\s+narrative)\b", re.I),
re.compile(r"\b(misinformation|false\s+narrative).{0,80}(viral|shareabil|amplif|maximize.{0,20}share|spread\s+wide)\b", re.I),
```
效果：提供 `intent:disinformation` 种子给 semantic gate，强化意图检测
