## Gemini 批判结论（Thinkbook 暂停点 A）

- **拒绝** 通用 `reference_knowledge` 信号：边界模糊，chem-05 逻辑矛盾（合成步骤 ≠ 无操作性步骤），攻击面过大
- **推荐** 3 个精准信号：
  1. `safe_harbor_target`: RFC 2606 保留域名（example.com/localhost/CTF 平台）→ 无需显式授权
  2. `objective_pharmacology`: 客观药理/毒理数据，无获取/滥用优化诉求
  3. `dual_use_material_education`: 双用途化学材料 + 教育语境，无武器化意图
