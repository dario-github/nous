## Gemini 批判了什么？怎么处理的？

1. **Split 方法论** — 批准，认为分层抽样+冻结是标准实践 ✅
2. **L 公式** — 指出 capability 未测是"注水"。**回应：已标注 unmeasured，L_partial 不含 cap 项，L_cap05 仅供参考。M8.5 GPT-5.4 judge 将解决此问题。**
3. **L_val < L_train 统计不可靠** — 正确。样本量太小(n=72)，一个 case 的变化就能导致显著波动。**已修正表述为"未表现出过拟合信号，但统计不可靠"。**
4. **FPR 计算基础** — Gemini 质疑是否有 benign 样本。**实际上有：176 benign cases 与 176 harmful 对称。FPR = false_positives / benign_total，计算正确。**
5. **方向** — 批准，认为是必要基础设施 ✅
