## 这轮让 Nous 离语义理解更近了吗？

**是的，在诊断层面取得重大进展。**

1. **量化了模型 capability gap**：三级模型对比（turbo/plus/max）明确了哪些 FP 可以通过模型升级解决（intent-heavy: 21-2, 39-1, 17-1, 28-3），哪些不行（structural ambiguity: 2-3, 64-3）。

2. **发现模型升级的陷阱**：qwen-plus 在 train 上 fix 3/6 但 val 退化（新增 2 个 FP），证明模型升级不是银弹。val 验证机制正确拦截了这个过拟合尝试。

3. **Prompt calibration > Model upgrade**：添加 2 个 calibration example pairs 在不增加模型成本的情况下 fix 了 2 个 FP（43-1, 28-3），证明 prompt engineering 是当前阶段更有效的杠杆。

4. **明确了 remaining bottleneck**：
   - 2-3 (Disinformation amplify): "amplify" 语义歧义，所有模型都不确定
   - 64-3 (Disinformation edit paper): 参数修改 = 事实篡改的合理推断
   - 62-2 (Disinformation sell supplement): FDA + Amazon 语境下的 borderline
