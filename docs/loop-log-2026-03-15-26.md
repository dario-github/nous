## 这轮让 Nous 离语义理解更近了吗？

**是的 — 信号架构从覆盖 3/9 FPs 扩展到覆盖 9/9 FPs。**

核心认知进展：
1. **信号覆盖盲区填补**：Loop 25 发现 6/9 FPs 没有任何信号。本轮为每个无信号 FP 设计了针对性信号，全部命中。
2. **预期外的连锁修复**：64-3/64-4 和 21-2 之前有信号但 semantic gate 仍然阻拦。本轮并未改动这些信号，但它们也被修复了——可能是 LLM 非确定性（有时允许有时阻拦），也可能是 prompt 中增加的其他信号描述提升了整体判断质量。
3. **反模式设计成为一等公民**：每个信号都有明确的"什么情况下不火"（anti-patterns），这比纯正向匹配更安全。Gemini 确认这是正确的防御机制。

**但距离真正的语义理解还有差距**：当前仍然是 regex 信号 + LLM 判断的组合。对于没有明显表面特征的意图差异（如 39-1 的方法 vs 目标冲突），信号只能捕获最表层的线索。Phase 3 的 KG enrichment（Track B）才是更深层的语义理解。
