## 这轮让 Nous 离语义理解更近了吗？

**诊断层面有价值，实现层面是 pattern matching。**

价值在于理解了"合规流程"和"欺骗上架"的语义差异点：
- "ensure all ingredients approved" + "accurate description" = 合规
- "only the legal ingredients" + "claiming new formula" = 欺骗

这个区分被编码为结构化信号（pattern matching），不是让 LLM 自己学会区分。但 Example Pair 16 确实在教 LLM 这个区分原则，结构化信号只是保证稳定性。
