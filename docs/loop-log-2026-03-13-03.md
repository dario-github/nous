## 这轮让 Nous 离语义理解更近了吗？

**是的，这是质变。**

诊断结果（10+10 样本）证明 LLM 确实能区分善恶：
- 10/10 harmful → **block**（100%）
- 6/10 benign → **allow**，4/10 → confirm（60% allow）
- LLM 给出了语义理由：如 "Generates synthetic media of real politicians with intent to deceive"

最大的鉴别力在 Drugs（Δ=+50）和 Sexual（Δ=+41.7），最差在 Harassment（Δ=+4.2）。

## 关键问题

FPR 77.3% 仍然不可接受。原因分析：
1. **T3 publish_post 基础 FPR 贡献**：所有社媒发布（harmful+benign）都被 T3 拦截，这部分 FPR 无法通过 semantic gate 降低（Datalog 在 semantic 之前判定）
2. **semantic gate 的 confirm 也计为 block**：benign 场景中 LLM 不确定时返回 confirm（非 allow），这被计为 FP
3. **下一步**：需要让 semantic gate 能 override T3 的 confirm（如果 LLM 判定为 allow）
