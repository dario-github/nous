## 做了什么

### 1. Signal Activation Map (Phase 1 Ablation)

Created `scripts/ablation_study.py` and ran Phase 1: signal activation mapping on both train and val sets. This is a zero-LLM-cost analysis that maps which of 13 structural signals fire on which scenarios.

**Train findings:**
- 12/13 signals active on at least one scenario
- Only `recipients_same_domain` has no `has_` prefix (it's domain-based)
- `has_ransom_signal` is the only harmful-detection signal (fires on 4 harmful)
- `has_academic_plagiarism_signal` also fires on harmful (1 case)
- All other signals fire on benign scenarios (FP reduction)

**Val findings:**
- 3 signals completely inactive: `educational_content`, `factual_correction`, `medical_wellness`
- These have zero val coverage → can't evaluate their contribution on val

### 2. Analytical Ablation (Signal ↔ FP Cross-Reference)

Cross-referenced signal activations with the 9 known train FPs:

| FP Scenario | Category | Signal Fired? | Status |
|-------------|----------|---------------|--------|

### 3. Gemini Critique (Hard Gate — Passed)

4. **Test set statistical power**: 36 benign cases → each error = ~3% FPR swing. Need challenge set urgently.

5. **Challenge set is the single most important Phase 3 item**.

All 5 critiques adopted into Phase 3 plan.
