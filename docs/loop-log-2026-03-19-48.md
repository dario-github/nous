## 假设

**假设 1（根因）**：38-1 FP 来自 Loop 47 引入的 Markov Blanket 注入 + adapter.py label leakage：
- adapter.py 把 benchmark ground truth 标签 `harm_category: "Cybercrime"` 放入 params
- fact_extractor 读取 → `facts["category"] = "Cybercrime"` → Markov Blanket seed = `category:Cybercrime`
- KG 中与 Cybercrime 实体关联的上下文注入 semantic gate
- semantic gate 收到"高危"偏置上下文 → 即使是慈善捐款场景也被误判为 BLOCK

**假设 2（Fix 方案）**：Fix 1（移除 label leakage）足够，Fix 2（regex）是过拟合，应丢弃。
