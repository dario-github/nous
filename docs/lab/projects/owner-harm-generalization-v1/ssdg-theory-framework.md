## 核心定义

- 攻击 = (意图, 表面, 上下文)
- 通用犯罪 A_G: 意图仅从表面可判定
- 主人伤害 A_O: 意图需要表面+上下文联合判定
- 防御信息集 Ω(D) = Ω_S(表面) ∪ Ω_C(上下文)
- 信息覆盖率 Cover(D,a) = |E(a)∩Ω(D)| / |E(a)|

## 三条公理

1. 检测-覆盖单调性：覆盖率越高，检测率越高
2. 完全覆盖上界：即使信息完全，检测率仍受推理能力限制
3. 不完全覆盖硬限制：覆盖率低于阈值θ时，检测率被硬性限制到ε

## 两个核心定理

定理1（符号防御的gap）：
- 符号防御 Ω_C = ∅（看不到上下文）
- 对 A_G：只需表面特征 → 覆盖率高 → 检测率高
- 对 A_O：需要上下文 → 覆盖率低于阈值 → 检测率崩溃
- 预测：巨大的 gap → 实测 100% vs 3.7% ✓

定理2（语义防御无gap）：
- 通用 LLM Ω_C = C_world（世界知识）
- 对两类攻击的覆盖率都由推理能力决定，不受结构性信息缺失影响
- 预测：gap很小 → 实测 62.7% vs 59.3% ✓

## 四个可验证预测

### P1: 上下文注入实验
向符号层注入显式上下文 → owner-harm TPR 提升 > 20pp
证伪：提升 ≤ 5pp

### P2: 语义层剥夺实验
剥夺语义防御的上下文 → 出现类符号 gap（gap比率 > 3倍）
证伪：gap比率 ≤ 1.5

### P3: 信息覆盖定律（S形曲线）
检测率与上下文比例呈 sigmoid 关系，非线性
- α < 0.2 时 TPR < 15%
- α > 0.8 时 TPR > 50%
- 中间存在陡峭相变
证伪：线性拟合 R² > 0.95

### P4: 跨类别迁移不对称
符号层：A_G → A_O 迁移增益 ≈ 0
语义层：A_G → A_O 迁移增益 > 10pp
证伪：符号层迁移 > 15pp

## 实验优先级

P1 和 P2 是核心（验证理论的两个方向：加信息 vs 减信息）
P3 如果做了会很漂亮但样本量要求高
P4 是补充性的
