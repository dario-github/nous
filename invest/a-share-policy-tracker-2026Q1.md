# 2026年A股政策风向与监管动态追踪

> 2026-04-16 整理 | 数据截至2026年4月中旬

## 一、量化交易监管：史上最严新规落地

### 核心政策：4月7日量化新规正式生效

| 维度 | 旧规 | 新规（2026.4.7生效） | 变化幅度 |
|------|------|---------------------|---------|
| 高频认定门槛 | ≥300笔/秒 | ≥15笔/秒 | **收紧20倍** |
| 撤单限制 | 无硬限制 | 每秒撤单≤15笔，单日撤单率≤15% | 新增 |
| 报单停留时间 | 无要求 | ≥50微秒 | 新增 |
| VIP独立通道 | 可专属使用 | 暂停新增，存量需≥10户共享 | 大幅收紧 |
| 穿透监管 | 无 | 配偶/父母/子女账户合并计算 | 新增 |

### 市场影响评估
- **流动性收缩**：预计日均成交额降15%-20%，小盘/微盘流动性萎缩20%-30%
- **波动收敛**：日内波动率从约3%降至约1%
- **策略洗牌**：几乎所有纯高频策略被纳入严管，速度型alpha策略失效
- **公平性提升**：散户面对的"秒挂秒撤"幌骗行为被基本杜绝

### 4月10日交易规则修订（征求意见）
- **盘后固定价格交易**扩至全A股及ETF（原仅限科创板/创业板）
- **ST/*ST涨跌幅**从5%调整为10%（征求意见中）
- 北交所增设风险警示股票单日买入数量上限

## 二、退市新规：退市节奏明显加快

### 2026年退市新特征
1. **退市概率飙升**：*ST股退市概率从约30%升至约70%
2. **缓冲期取消**：触发退市条件后直接进入15个交易日整理期（原流程缩短>50%）
3. **市值退市成主流**：2026年至今市值退市案例已超过面值退市
4. **45家公司年报后或被ST**：总市值1287亿，涉及股东>156万户

### 已退市/即将退市案例（2026年至今）
- **长药控股**：财务造假退市（4月13日摘牌）
- **ST奥维**：首家市值退市公司（1月退市）
- 广道退、东通退、立方退等多只股票退市或即将退市

### 对投资策略的影响
- **壳资源价值归零加速**：小市值"炒壳"策略风险极端放大
- **选股必须避开ST预警区**：需建立财务红线筛查机制
- **微盘股流动性溢价消退**：退市常态化压低垃圾股估值

|------|------|

| 首发过会 | 23家（3月单月） |

### 板块格局
- **北交所占比超六成**：在审企业188家/总量300家（63%）
- 硬科技企业成募资主力
- IPO标准已细化至16套，各板块定位清晰

### 趋势判断
- IPO节奏"稳起步、快收尾"，审核效率提升
- 注册制深化，但质量把关不放松
- 大盘股供给压力可控，北交所扩容是主旋律

## 四、北向资金：外资态度分化

### 一季度数据（存在不同口径差异）
- **口径1（沪深股通持股数据）**：一季度净流出142亿元，持股市值2.58万亿元（较Q4基本持平）
- **口径2（含ETF通道）**：一季度净流入3274亿元，创历史同期最高，外资借道ETF买入近600亿元

> 注：两个口径差异主要来源于统计口径不同（是否含ETF、是否含QFII直投等），但共识是外资对A股结构性看好。

### 持仓特征
- **前十大重仓**：宁德时代、贵州茅台、美的集团、招商银行、中际旭创、北方华创、紫金矿业、汇川技术、思源电气、中国平安
- **重点增持方向**：光模块三杰（中际旭创/天孚通信/新易盛）加仓超231亿元
- **行业偏好**：电池、半导体、通信设备（持仓市值均>1100亿元）
- **外资机构态度**：摩根士丹利看好原材料/工业品/半导体/能源；瑞银看好AI领域低估值机会

### 政策动向
- 证监会2026年立法计划中持续优化外资准入
- 中长期资金入市机制进一步完善
- 构建中国特色稳市机制，外资政策基调偏积极

## 五、微盘股/小市值监管态度

### 监管定位
- **非一刀切打压**：监管针对的是欺诈/操纵/退市红线，并非市值本身
- **支持科创小微重组并购**：2024.5并购新规后，大型企业并购微盘股案例未见明显下降
- **北交所设置买入上限**：风险警示股票单日买入数量受限，引导理性投资

### 风险提示
- 微盘股在量化新规后**流动性萎缩更显著**（20%-30%）
- 壳资源逻辑被退市新规严重削弱，但"被并购"仍是微盘股的阶段性估值支撑
- 地缘政治敏感期，微盘股波动放大

### 对量化策略的影响
| 策略类型 | 影响 |
|---------|------|
| 高频做市 | 几乎无法运行 |
| 微盘股Alpha | 流动性大幅下降，滑点成本飙升 |
| 退市风险规避 | 必须纳入财务红线筛选 |
| 事件驱动（并购） | 仍有空间，但需精细筛选 |

## 政策影响综合评估

### 利好因素 ✅
1. 量化新规提升市场公平性，散户交易环境改善
2. 退市常态化净化市场生态，资源向优质企业集中
3. 外资结构性流入，科技板块获重点配置
4. 盘后交易扩至全A股，增加交易灵活性

### 风险因素 ⚠️
1. 量化新规导致短期流动性收缩，小盘股冲击更大
2. ST涨跌幅放宽至10%后，风险警示股波动加剧
3. 退市加速带来踩雷风险（45家或被ST）
4. IPO供给恢复可能分流存量资金

### 策略建议
- **回避**：纯高频策略、壳资源炒作、ST/*ST投机
- **关注**：大盘蓝筹（外资流入受益）、科技板块（半导体/AI/光模块）、并购重组机会
- **风控**：建立退市预警筛查机制，对持仓做财务红线过滤
