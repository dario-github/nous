# 私募知识突击 — 4周入职准备

> 背景：东丞确认入职中诚志远私募（北京中诚志远投资有限公司，AMAC: P1067541），AI负责人，5/15 正式入职。团队 AUM 7亿+。
> 目标：4周内建立私募行业认知框架 + 明确 AI 负责人的价值创造路径。

## Week 1 (04-15 → 04-21): 行业全景 + 监管框架

### 目标
建立对中国私募行业的系统性认知，理解监管环境和合规底线。

### 学习清单
- [ ] **行业概览**：中基协数据（私募规模、管理人数量、策略分类、存活率）
- [ ] **监管体系**：AMAC 备案流程、合格投资者认定、信息披露要求、托管制度
- [ ] **策略分类**：股票多头、量化对冲、CTA、固收+、FOF、宏观对冲
- [ ] **7亿规模定位**：mini fund 的优势（灵活、集中持仓）与劣势（费率议价弱、渠道受限）
- [ ] **AI在私募的现状**：量化已成熟，非量化私募的AI渗透率仍低（机会所在）

### 产出
- [x] 行业认知笔记 → vault/知识库/concepts/中国私募行业概览.md ✅ 2026-04-17 heartbeat 完成
- [ ] 监管红线清单（AI负责人必须知道的合规底线）

## Week 2 (04-22 → 04-28): 投研流程 + AI切入点

### 目标
理解私募投研全流程，识别 AI 可以创造价值的环节。

### 学习清单
- [ ] **投研流程**：宏观→行业→个股→组合构建→风控→交易执行
- [ ] **研究方法**：自上而下 vs 自下而上、基本面 vs 量化
- [ ] **风控体系**：回撤控制、仓位管理、行业集中度、个股集中度
- [ ] **AI 具体应用场景**：
  - 研报自动摘要 + 观点提取
  - 舆情监控 + 异常信号
  - 财报数据自动结构化
  - 投资组合归因分析
  - 合规文档审查
- [ ] **竞品调研**：国内私募AI应用案例（幻方、九坤、明汯等头部量化的AI基础设施）

### 产出
- AI应用优先级矩阵（价值 × 实现难度）
- 3个可快速落地的AI方案草案

## Week 3 (04-29 → 05-05): 技术栈 + 实施路径

### 目标
设计技术架构，明确可复用的现有能力 vs 需要新建的。

### 学习清单
- [ ] **私募IT基础设施**：交易系统（恒生/顶点）、数据源（Wind/Choice/Tushare）、合规系统
- [ ] **东丞现有能力盘点**：
  - 内容宇宙 AI 平台经验（蓝标积累）
  - 知识图谱（同花顺积累）
  - Agent 系统（OpenClaw 生态）
  - LLM 应用开发
- [ ] **技术选型**：自建 vs SaaS、开源 vs 商用、云端 vs 本地
- [ ] **数据合规**：数据出境、个人信息保护、交易数据留存

### 产出
- 技术架构草图
- 预算估算（人月/云资源/数据源）
- 90天实施路线图

## Week 4 (05-06 → 05-12): 入职准备 + 30天计划

### 目标
准备好第一天就能创造价值的所有材料。

### 学习清单
- [ ] **组织理解**：团队结构、汇报关系、决策流程（入职后再补全）
- [ ] **30/60/90天计划**：
  - 30天：理解业务 + 快速 wins（现有流程优化）
  - 60天：第一个AI产品上线（如研报助手）
  - 90天：基础设施搭建完成，开始规模化
- [ ] **沟通准备**：如何向非技术合伙人解释AI价值
- [ ] **风险预判**：技术债务、团队技能缺口、数据质量问题

### 产出
- 30/60/90天计划文档
- 第一周行动清单
- 需要确认的问题清单（入职后问）
