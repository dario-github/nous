# Nous Invest — System Specification v0.3
## 2026年量化选股系统：从"同款"到"缝隙"

---

## 1. 愿景

构建个人/小资金专属的量化选股系统，利用**容量缝隙**和**非同质化信号**，在2026年量化拥挤环境中获取可持续超额。

---

## 2. 核心原则

1. **不自研基础设施** — 只使用成熟框架（Qlib、Tushare、AkShare）
2. **追求周超额而非日IC** — 评估周期对齐私募标准
3. **容量优先** — 小市值、非机构拥挤区
4. **信号独特性** — 避开Alpha158等"同款"因子
5. **频段错配** — 高/中/低频信号叠加

---

## 7. 技术栈

| 组件 | 选择 | 理由 |

## 9. 自动推进规则

1. **每日**: 自动下载数据 → 生成信号 → 输出报告
2. **每周**: 周超额评估 → 模型性能分析 → 参数调优
3. **每月**: 策略回顾 → 信号独特性检查 → 路线图更新
4. **每季**: 大版本迭代 → 新模块上线

---

## 10. 成功标准 (Exit Criteria)

系统达到以下标准即视为"生产就绪":

- [ ] 连续4周周超额 > 0.5%
- [ ] 周超额夏普 > 1.5
- [ ] 最大回撤 < 15%
- [ ] 信号与Alpha158相关性 < 0.5
- [ ] 容量估计 > 500万
- [ ] 全自动运行无人工干预
